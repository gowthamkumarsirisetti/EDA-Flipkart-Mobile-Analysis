  <h1>📊 Flipkart Mobile EDA Project</h1>

  <div class="section">
    <h2>🛍️ Use Case</h2>
    <p>Flipkart hosts hundreds of smartphones. Customers struggle to compare features, and businesses need data-backed insights. This project uses Python to scrape and analyze mobile data from Flipkart for trends in pricing, RAM, battery, reviews, and more.</p>
  </div>

  <div class="section">
    <h2>🎯 Objectives</h2>
    <ul>
      <li>Scrape 200+ pages of smartphone listings using <code>requests</code>, <code>BeautifulSoup</code>, and <code>regex</code></li>
      <li>Clean and structure data using <code>pandas</code></li>
      <li>Perform EDA to uncover market trends and patterns</li>
      <li>Help customers and sellers make data-driven decisions</li>
    </ul>
  </div>

  <div class="section">
    <h2>📦 Files Included</h2>
    <ul>
      <li><strong>Group Project Flipkart.ipynb</strong> – Main EDA notebook</li>
      <li><strong>EDA Project PPT on Flipkart.pptx</strong> – Presentation slides</li>
      <li><strong>flipkart_mobile_data.csv</strong> – Dataset used (optional)</li>
      <li><strong>README.html</strong> – This project overview page</li>
    </ul>
  </div>

  <div class="section">
    <h2>🔍 Key Insights</h2>
    <ul>
      <li>Most mobiles are priced between ₹10K–₹20K — the most competitive segment</li>
      <li>5000 mAh battery and 8GB RAM are most common</li>
      <li>16GB RAM phones are premium and highest rated</li>
      <li>Discounts between 15%–35% are standard</li>
      <li>Top brands: Samsung, Realme, Xiaomi</li>
    </ul>
  </div>

  <div class="section">
    <h2>💡 Conclusion</h2>
    <p>Mid-range mobiles dominate the market. Battery size and RAM strongly influence customer reviews. Brand strength varies with price and features. Real-world EDA helped us extract business-level insights from messy web data.</p>
  </div>
